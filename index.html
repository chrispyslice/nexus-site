Nexus a new look at content management systems

Welcome to Nexus, a new look at how we think of content management systems. Nexus is designed to redefine the CMS by taking a document-orientated approach, instead of a system-oriented approach like other content management systems.

Nexus is written in PHP and uses the CodeIgniter framework.

You can also check out or GitHub page.

Contributing

You don't need to know PHP or CodeIgniter to contribute. There are many other ways to contribute to Nexus:

  • Download the code and report bugs on our SourceForge tracker
  • Submit feature requests
  • Help with the documentation

Documentation

At the moment, documentation is a little patchy as Nexus is in pre-alpha. The most current documentation is available on our wiki at GitHub.

Download

The best way of downloading Nexus to to checkout the latest commit on GitHub using: git clone git://github.com/chrispyslice/nexus Nexus is currently in pre-alpha status, so things probably won't work. However, I am trying to get things working as quickily as possible, so be pateint.

The current requirements for Nexus are as follows (although this list may not stay updated, if you have these Nexus will probably work):

  • PHP 4.3.2 or newer (5.2 or above is recommended)
  • Nexus comes with built-in support for many databases, including:
    • MySQL and MySQLi
    • Microsoft SQL Server
    • PosgreSQL
    • SQLite
    • Oracle
    • Open Database Connectivity
    See Databases on our wiki for more details.

Support

There isn't really much point in asking for support at this point because Nexus is so incomplete. However, if you really feel the need to do so, support is available on our Support Tracker over at SourceForge.
